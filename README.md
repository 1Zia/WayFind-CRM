# WayFind

Base Next.js 14 App Router scaffold with TypeScript, TailwindCSS, and placeholders for:

- Clerk auth
- Drizzle ORM + Neon
- Zod validation
- shadcn/ui components
- Lucide icons
- Sonner toasts

Next steps:

1. Run `npm install` to install dependencies.
2. Configure environment variables (Clerk, Neon connection).
3. Implement auth and DB integration.
